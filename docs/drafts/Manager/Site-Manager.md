## Quick Start New Task
This quickstart guide takes the mananger through the process of mananging tasks for site inspections.

1. As supervisor/site manager login to web console
2. Select a user and assign / create a new task
3. Task creation select a *Purpose* for which the inspection, this is very important step, as later will be able filter and sort by *Purpose*
4. The inspector will have a new Task listed in Mobile app and Web Console

## Quick Start Task Actions
5. Supevisor/ site manager reviews photos taken during inspections.
6. Change *Status* of Task, Approves, or returns to inspection to caputure additional photos.
7. PDF report of site inspection is created http://preview.pic2bim.co.uk/Inspector/Web-Console/?h=pdf#411-generating-pdf-documents

### User roles permissions

* Have access to his tasks and photos, but is not able to do any operations.

#### Officer (Adminstrator) role

* Have access to all users that belong under his agency, can move tasks between states.

#### Super-admin role

* Have access to Agency management module. Can create new agencies and
    manage existing ones.

* Does not have access to standard application.
